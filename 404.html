Where is my data stored?

All your data lives in this phone's browser storage (called "localStorage"), tied to this specific browser app/tab. It is private to this device and survives closing the app, restarting your phone, etc.

It can be lost if: you clear your browser's site data/cache for this app, uninstall/reinstall the browser, or use a different browser or device.

To keep it safe: download a backup (.json) regularly using the button below, and/or connect Google Sheets above for an automatic cloud copy.

📖 Google Sheets Setup

1. Go to sheets.google.com - create a sheet named Laasya Dance.
2. Click Extensions → Apps Script.
3. Delete all code, paste the script (button below).
4. Click Deploy → New deployment → Web app. Set access to Anyone. Copy the URL.
5. Paste that URL in Settings → Google Sheets → Edit.
